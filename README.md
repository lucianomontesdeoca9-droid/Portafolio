# Portafolio — Mi portafolio AI

Un portfolio web moderno y responsivo para mostrar tus proyectos y habilidades de manera profesional.

## 🚀 Características

- Diseño moderno y limpio
- Modo oscuro/claro (opcional)
- Totalmente responsivo (móvil, tablet, escritorio)
- Animaciones y transiciones suaves
- Secciones para proyectos, habilidades y contacto
- Enlaces a redes sociales

## 📋 Secciones

1. **Inicio** — Presentación con links y llamada a la acción
2. **Sobre Mí** — Biografía breve y habilidades
3. **Habilidades** — Tecnologías y herramientas
4. **Proyectos** — Trabajos destacados con imágenes
5. **Contacto** — Formulario y datos de contacto

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (Variables CSS, Grid, Flexbox)
- JavaScript (Vanilla)

## 📦 Cómo usar

1. Clona el repositorio:
```bash
git clone https://github.com/tuusuario/portfolio.git
```

2. Personaliza tu información en [index.html](index.html)

3. Ajusta estilos en [styles.css](styles.css)

4. Reemplaza imágenes y contenido de ejemplo

## 🎨 Personalización de colores

Las variables de color están en [styles.css](styles.css):

```css
:root {
  --primary-color: #667eea;
  --secondary-color: #764ba2;
}
```

## 📱 Responsivo

Optimizado para móviles (<768px), tablets (768–1024px) y escritorio (>1024px).

## 🚀 Despliegue en GitHub Pages

1. Ve a tu repositorio en GitHub → Settings → Pages
2. Selecciona la rama `main` como Source
3. Guarda y espera unos minutos

## ✅ Checklist de personalización
